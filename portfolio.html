  • Hash Table for Managing Comedian Data

    Technologies: C++ | Role: Solo Project | Duration: Aug. 2024

    This project is a C++ implementation of a hash table designed to manage information about comedians from the show "Kill Tony." The program allows users to add, find, delete, display, and save comedian data efficiently using a hash table.

    Key Features:

    • Hashing: Utilizes a hash function to efficiently store and retrieve comedian data based on a unique key.
    • Linked Lists: Each slot in the hash table is implemented as a linked list to handle collisions, ensuring data integrity even when multiple entries hash to the same slot.
    • Data Management: Users can track multiple appearances of comedians, including the episodes they've appeared on and any associated notes.
    • File I/O: The program supports saving the current state of the hash table to a file and loading data from a file, with entries saved in a structured format.

    This project showcases the effective use of data structures and file handling in C++ to manage and manipulate complex datasets.
